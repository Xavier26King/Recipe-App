# Project Setup

## Frontend

### Installation
1. Navigate to the `frontend` directory:
   ```bash
   cd client
````

2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the development server:

   ```bash
   npm run dev
   ```

## Backend

### Installation

1. Navigate to the `recipe-backend` directory:

   ```bash
   cd backend
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the server using `nodemon`:

   ```bash
   nodemon
   ```
